# 🏆 UNO Live Scoreboard

A real-time UNO game scoreboard for tracking matches between players.

## Features
- 📊 Live score tracking
- 🔄 Real-time updates (10-second sync)
- 👤 Admin panel for managing rounds
- 📱 Responsive mobile design
- 🎨 UNO-themed UI

## Usage

1. Open `index.html` in your browser
2. Click "ADMIN LOGIN" (bottom left, very faint text)
3. Enter password: `uno123`
4. Select winner and enter points
5. Click "Post to Live"
